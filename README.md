trainer

2. 탭을 이용하여 클릭이벤트를 주게되어 개인페이지로 이동 구현
3. 각 트레이너들의 개인 페이지에 트레이너 개인 회원관리에 대한 글과 소개글을 작성할수 있는 공간 구현

2. 주로 이미지보다는 사용방식을 표현하기 위해 간단한 사용방법 텍스트와 동영상으로 구현


photo
